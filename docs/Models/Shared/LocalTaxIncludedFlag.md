# LocalTaxIncludedFlag

Provides details about the tax amount.<br>
Valid values:<br>
<ul>
  <li>0: Sales Tax not Provided</li>
  <li>1: Local or Sales Tax Amount</li>
  <li>2: Tax Exempt</li>
</ul>



## Values

| Name   | Value  |
| ------ | ------ |
| `Zero` | 0      |
| `One`  | 1      |
| `Two`  | 2      |
